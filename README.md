# GKhere-Android
广科这里 - 校园客户端  
## 应用介绍
GKhere是广东科技学院Android客户端,仅限广东科技学院学生使用,遵循Material Design.

## 已实现功能
- 查课表
- 查成绩
- 图书馆找书
